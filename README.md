# 🛍️ Vintage Collection Catalog

이 프로젝트는 구글 스프레드시트와 구글 드라이브를 연동하여 만든 **빈티지 의류 카탈로그 웹사이트**입니다.
[Streamlit](https://streamlit.io/)을 사용하여 제작되었습니다.

## 🌟 주요 기능
- **상품 조회**: 구글 시트에 등록된 상품 정보를 실시간으로 불러옵니다.
- **필터링**: 브랜드, 카테고리, 사이즈, 가격 범위, 품절 여부 등으로 필터링할 수 있습니다.
- **검색**: 상품명 키워드로 검색이 가능합니다.
- **이미지**: 구글 드라이브에 저장된 이미지를 자동으로 불러옵니다.
- **구매 문의**: 카카오톡 오픈채팅 또는 전화 연결 버튼을 제공합니다.

## 🛠️ 설치 및 실행 (로컬)

1. **파이썬 설치**: Python 3.8 이상이 필요합니다.
2. **라이브러리 설치**:
    ```bash
    pip install -r requirements.txt
    ```
3. **설정 파일**:
    - `.streamlit/secrets.toml` 파일에 구글 서비스 계정 키(`gcp_service_account`)가 필요합니다.
4. **실행**:
    ```bash
    streamlit run main.py
    ```

## 📂 파일 구조
- `main.py`: 메인 화면 및 UI 로직
- `data_loader.py`: 구글 시트 데이터 로딩 및 이미지 처리
- `requirements.txt`: 필요한 파이썬 라이브러리 목록
